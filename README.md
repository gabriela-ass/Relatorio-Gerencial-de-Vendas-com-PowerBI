
# 📊 Power BI Financial Analysis
Este repositório contém o projeto de Business Intelligence desenvolvido durante o bootcamp da DIO. O objetivo foi criar um relatório dinâmico e intuitivo utilizando a base de dados Sample Financials, focando em usabilidade (UX), navegabilidade e análise de performance financeira.


## 🚀 Tecnologias Utilizadas
- Power BI Desktop
- Power Query (Processamento e transformação de dados)
- GitHub (Versionamento de projeto)

## 📈 Funcionalidades do Relatório
O relatório foi estruturado para oferecer uma visão clara dos indicadores financeiros, permitindo que o usuário navegue entre diferentes níveis de detalhamento.

1 - Estrutura e Layout
Design Customizado: Layout planejado para facilitar a leitura dos dados.

- Navegação Dinâmica: Utilização de botões de navegação para alternar entre as páginas do relatório.

- Interatividade: Botões com imagens associadas e segmentadores de dados (filtros) que permitem análises por período, produto e região.

2 - Elementos Técnicos Aplicados
Indicadores de Performance: Uso de cartões para métricas principais (Vendas Totais, Lucro, Unidades Vendidas).

- Visualizações Avançadas: Gráficos de barras, linhas e pizza configurados para destacar tendências e outliers.

- Bookmarks (Indicadores): Configuração de botões para alternar visuais sobre um mesmo assunto, otimizando o espaço da tela.

## 📂 Estrutura das Páginas
Página 1 (Home/Visão Geral): Focada nos principais KPIs e navegação inicial.

Página 2 (Detalhes/Análise de Produto): Um mergulho mais profundo nos dados, explorando a performance por categoria e país.

### 🔗 Como visualizar o projeto
#### Faça o download do arquivo .pbix presente neste repositório.
